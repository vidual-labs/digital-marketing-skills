# TikTok Ads Creatives & Copy

## Overview

Create high-converting creative concepts and ad copy for TikTok advertising. TikTok demands a fundamentally different creative approach than Meta: native, fast, authentic, and entertainment-first. This skill covers hook formulas, script structures, in-app ad copy, campaign formats, and testing frameworks.

## When to Use

- Writing TikTok ad scripts and concepts
- Developing creative strategy for TikTok campaigns
- Adapting existing Meta/YouTube creative for TikTok
- Brainstorming hooks, trends, and formats for TikTok Ads Manager
- Creating in-app ad copy (primary text, headlines, descriptions)

Don't use for: TikTok organic content strategy, TikTok Shop seller strategy, or UGC creator outreach scripts.

## TikTok In-App Ad Fields

| Field | Placement | Character Limit | Purpose |
|-------|-----------|----------------|---------|
| **Primary Text** | Above video | 100 chars visible (150 max) | Short hook + value prop |
| **Headline** | Next to CTA button | 40 chars visible (50 max) | Reinforce offer or urgency |
| **Description** | Below headline | 60 chars visible (2000 max but truncated) | Supplementary detail — rarely read |
| **CTA Button** | Next to headline | Fixed set | Action button |
| **Video (creative)** | Main content | 4s-60s (optimal 15-30s) | The actual ad — does the heavy lifting |

**Key difference from Meta:** TikTok primary text is much shorter. The video creative carries 80% of the lift. Copy supports; it doesn't lead.

## Creative Philosophy: "Native First"

TikTok users aggressively tune out anything that looks like an ad. Creative must feel native — like organic content — or it fails.

| Do | Don't |
|----|-------|
| Film vertically (9:16), shot on phone | Film in 16:9 or overly polished studio style |
| Start with movement/action in frame 1 | Start with a logo or brand shot |
| Speak naturally, with pauses and mistakes | Use a corporate voiceover or overly scripted tone |
| Use trending audio or original sound | Use generic stock music |
| Show real products in real environments | Use white-background product shots |
| Hook in the first 1-2 seconds | Spend first 3 seconds building context |

## Hook Formulas

The first 2 seconds determine whether the video lives or dies. TikTok gives 2 seconds of auto-play before the user scrolls — the hook must win in that window.

### 1. Statement Hook
- "The biggest mistake [audience] makes with [topic]"
- "Stop [common action]. Do this instead."

### 2. Curiosity Hook
- "I tried [thing] for [time] and [unexpected result]"
- "Nobody talks about this [industry] problem"

### 3. Result Hook
- "How I [specific result] in [timeframe]"
- "[Number] [people] use [thing] wrong — here's the fix"

### 4. Demo Hook
- Show the product doing something impressive immediately
- "Watch this in 5 seconds"

### 5. Story Hook
- "So I [did something] and it [happened]"
- "My [friend/client] used to [problem] until this"

### 6. Trend Hijack
- Use trending audio/template but adapted to your niche
- Must be within the first 3 days of the trend's peak

### 7. Pattern Interrupt
- Unexpected visual: upside-down camera, sudden zoom, weird angle
- Visual weirdness stops the scroll before words even matter

**Hook rules:**
- No greeting: never say "hey guys" or "what's up"
- No intro: "I'm John from X company" is a scroll-away
- Front-load: the hook line is also the first line on screen as text
- On-screen text is mandatory — 50% of users watch muted initially

## Video Script Structure

### 15-Second Format (Top-of-funnel / awareness)

```
0-2s  HOOK — visual + text-on-screen + spoken
2-5s  PROBLEM — state it fast, one sentence
5-10s SOLUTION — show the product/method in action
10-15s CTA — "Link in bio" / button prompt
```

### 30-Second Format (Mid-funnel / conversion)

```
0-2s   HOOK — stop the scroll
2-7s   PROBLEM — specific, relatable, personal
7-12s  AGITATE — why it costs them (time, money, frustration)
12-20s SOLUTION — show transformation or demo
20-26s PROOF — "This helped [X people get Y result]"
26-30s CTA — match the button text, create urgency
```

### 45-60-Second Format (Bottom-funnel / complex offers)

```
0-2s    HOOK
2-10s   PROBLEM / STORY — longer setup, personal anecdote
10-20s  SOLUTION — step-by-step demo
20-35s  PROOF — testimonial voiceover, real results, before/after
35-45s  OBJECTION HANDLING — address the #1 reason people don't buy
45-60s  CTA + URGENCY — "Only X spots left" / "Ends Sunday"
```

## TikTok Ad Formats

| Format | Description | Best For |
|--------|-------------|----------|
| **In-Feed Video** | Full-screen vertical, auto-play in For You feed | Default format, most campaigns |
| **TopView** | First impression when user opens app | Big launches, brand awareness |
| **Branded Effects** | Custom AR filters / stickers | Younger audiences, interactive brands |
| **Branded Hashtag Challenge** | Sponsored UGC campaign | Mass awareness, viral potential |
| **Branded Series** | Episodic content on creator profiles | Story-driven campaigns |
| **Collection** | Video + shoppable carousel | E-commerce with multiple products |

| Funnel Position | CTA |
|----------------|-----|

| Middle-funnel (consideration) | Sign Up, Learn More |
| Bottom-funnel (conversion) | Shop Now, Apply Now, Get Quote |

## TikTok-Specific Copy Rules

1. **Write like you talk.** TikTok copy that reads like a press release dies. Contractions, fragments, casual language — all are encouraged.

2. **On-screen text is mandatory.** 50% of viewers watch muted. Every key point should appear as text overlay. Use the native TikTok text style (not external graphics).

3. **Caption + primary text synergy.** Primary text above the video teases the hook. The caption (below video) can be longer — use it for storytelling or additional context.

4. **Use trending phrases sparingly.** "This is so [emotion] for [use case]" or "POV:" format can work but dates quickly. Trending language has a 5-7 day half-life.

5. **First comment matters.** Pin a comment with your CTA, link, or additional info. Comments are highly visible on TikTok.

## Creative Testing Strategy

| Test | What to Change | How to Isolate |
|------|---------------|----------------|
| Hook test | First 2 seconds only | Same video body, different opening |
| Format test | 15s vs 30s vs 60s | Same concept, cut to different lengths |
| CTA test | Button + verbal CTA | Same creative, different CTA text |
| Visual style | UGC vs brand-produced | Same script, different production |
| Audio test | Trending music vs original vs VO | Same visual, different audio |

**TikTok Creative Center:** Use TikTok's built-in creative benchmark tool to see what's working in your industry before filming.

## Common Pitfalls

1. **Porting Meta creative to TikTok without changes.** Meta creative is too polished, too long, and too "ad-like" for TikTok. Re-shoot native.

2. **Skipping the hook.** First 2 seconds are the only thing that matters. If the hook doesn't stop the scroll, the best product in the world won't help.

3. **White-background product shots.** These get scrolled past instantly. Show the product in a real setting, in real hands.

4. **Over-producing.** TikTok's top-performing ads often look like they were shot on an iPhone with zero editing. Higher production ≠ higher performance.

5. **Not using on-screen text.** Half of your audience is muted. No text = no message = wasted impression.

6. **Ignoring TikTok's comment culture.** Top comments can drive engagement 5× higher than the video itself. Pin strategic comments.

7. **Too many ads with the same hook.** TikTok's algorithm is good at detecting repeated creatives and throttling them. Rotate hooks weekly.

## Verification Checklist

- [ ] At least 3 creative variations with different hook types
- [ ] Hooks under 2 seconds (first line of spoken + visual)
- [ ] On-screen text included for every section
- [ ] Primary text under 100 characters
- [ ] Headline under 40 characters and complements the video
- [ ] CTA button matches funnel position
- [ ] Script broken down second-by-second with visual direction
- [ ] Creative feels native (not like a TV ad)
- [ ] Trending audio considered (if applicable)
- [ ] Comment strategy included (pinned comment)
